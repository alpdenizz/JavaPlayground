####################### READ ME ##############################

1) gtfs directory must be at the same filepath with java files

2) You can execute the code with:
	
	javac Boundary.java
	
	java Boundary departure_id arrival_id date time

For example:

	java Boundary 3894 12027 2017-09-10 15:00

	java Boundary 830 1202 2017-10-15 16:00

NOTE: Date and Time must be written in this format; otherwise, the code will not work properly.

Date Format: yyyy-MM-dd

Time Format: HH:mm
